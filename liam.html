Cloud Computing

Introduction

Cloud computing is a rather extensive topic in computation application. On a basic level, it opens a vast number of opportunities for sharing a range of resources and hardware. These resources are accessible from a local machine which comes with many benefits both to individuals and businesses.

What is the cloud?

The cloud could be used for many different applications within computing. The most know usage would be for cloud storage. Being able to store local files remotely has great potential. A great concern for a business is hardware failure. Should a device or machine fail or break, it is very likely for the data stored on that machine to be lost too. Of course, this is unlikely. It is common practice to back up any data to prevent it from being lost completely. Cloud storage gives a business or individual the change to store data remotely.
